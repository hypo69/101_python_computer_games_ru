**–û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏:**

    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê}`, `B = {üçê, üçâ}`. `A ‚à™ B = {üçé, üçê, üçâ}`.

2.  **–ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ (Intersection):** –ü–æ–∏—Å–∫ –æ–±—â–∏—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –¥–≤—É—Ö –º–Ω–æ–∂–µ—Å—Ç–≤–∞—Ö. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –±—É–¥–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ —Ç–æ–ª—å–∫–æ —Ç–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ –æ–±–æ–∏—Ö –º–Ω–æ–∂–µ—Å—Ç–≤–∞—Ö.

    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê}`, `B = {üçê, üçâ}`. `A ‚à© B = {üçê}`.

3.  **–†–∞–∑–Ω–æ—Å—Ç—å (Difference):** –ù–∞—Ö–æ–∂–¥–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ –ø–µ—Ä–≤–æ–º –º–Ω–æ–∂–µ—Å—Ç–≤–µ, –Ω–æ –æ—Ç—Å—É—Ç—Å—Ç–≤—É—é—Ç –≤–æ –≤—Ç–æ—Ä–æ–º. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –±—É–¥–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ –≤ –ø–µ—Ä–≤–æ–º –º–Ω–æ–∂–µ—Å—Ç–≤–µ.

    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê}`, `B = {üçê, üçâ}`. `A - B = {üçé}`, `B - A = {üçâ}`.

4.  **–°–∏–º–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å (Symmetric Difference):** –ù–∞—Ö–æ–∂–¥–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ –≤ –æ–¥–Ω–æ–º –∏–∑ –º–Ω–æ–∂–µ—Å—Ç–≤, –Ω–æ –Ω–µ –≤ –æ–±–æ–∏—Ö. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –±—É–¥–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –µ—Å—Ç—å –≤ A –∏–ª–∏ –≤ B, –Ω–æ –Ω–µ –≤ –æ–±–æ–∏—Ö –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ.
    *   –û–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º `^` –∏–ª–∏ –º–µ—Ç–æ–¥–æ–º `symmetric_difference()`.
    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê}`, `B = {üçê, üçâ}`. `A ^ B = {üçé, üçâ}`.

5.  **–ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Subset):** –ü—Ä–æ–≤–µ—Ä–∫–∞, —è–≤–ª—è–µ—Ç—Å—è –ª–∏ –æ–¥–Ω–æ –º–Ω–æ–∂–µ—Å—Ç–≤–æ —á–∞—Å—Ç—å—é –¥—Ä—É–≥–æ–≥–æ. –ú–Ω–æ–∂–µ—Å—Ç–≤–æ A —è–≤–ª—è–µ—Ç—Å—è –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º B, –µ—Å–ª–∏ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã A —Å–æ–¥–µ—Ä–∂–∞—Ç—Å—è –≤ B.
    *   –û–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º `<=` –∏–ª–∏ –º–µ—Ç–æ–¥–æ–º `issubset()`.
    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê}`, `B = {üçé, üçê, üçâ}`. `A <= B` (A —è–≤–ª—è–µ—Ç—Å—è –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º B).

6.  **–ù–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Superset):** –ü—Ä–æ–≤–µ—Ä–∫–∞, —Å–æ–¥–µ—Ä–∂–∏—Ç –ª–∏ –æ–¥–Ω–æ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –¥—Ä—É–≥–æ–µ. –ú–Ω–æ–∂–µ—Å—Ç–≤–æ A —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º B, –µ—Å–ª–∏ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã B —Å–æ–¥–µ—Ä–∂–∞—Ç—Å—è –≤ A.
    *   –û–±–æ–∑–Ω–∞—á–∞–µ—Ç—Å—è —Å–∏–º–≤–æ–ª–æ–º `>=` –∏–ª–∏ –º–µ—Ç–æ–¥–æ–º `issuperset()`.
    *   –ü—Ä–∏–º–µ—Ä: `A = {üçé, üçê, üçâ}`, `B = {üçé, üçê}`. `A >= B` (A —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ–º B).

**Python Code (—Å –ø–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ–º –∏ —Ä–∞–∑—ä—è—Å–Ω–µ–Ω–∏—è–º–∏):**

```python
from typing import Set

def create_fruit_set(fruit_string: str) -> Set[str]:
    """
    –°–æ–∑–¥–∞–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤ –∏–∑ —Å—Ç—Ä–æ–∫–∏.

    Args:
        fruit_string: –°—Ç—Ä–æ–∫–∞ —Å —Ñ—Ä—É–∫—Ç–∞–º–∏ (üçé, üçê, üçâ, üß∫).

    Returns:
        –ú–Ω–æ–∂–µ—Å—Ç–≤–æ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —Ñ—Ä—É–∫—Ç–æ–≤.
    """
    if not all(fruit in ["üçé", "üçê", "üçâ", "üß∫"] for fruit in fruit_string):
        raise ValueError("–°—Ç—Ä–æ–∫–∞ –º–æ–∂–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å —Ç–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª—ã üçé, üçê, üçâ, üß∫")
    return set(fruit_string)  # –ò—Å–ø–æ–ª—å–∑—É–µ–º set() –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞

def display_set(fruit_set: Set[str]) -> str:
  """
  –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤ –≤ —Å—Ç—Ä–æ–∫—É –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è.

    Args:
        fruit_set: –ú–Ω–æ–∂–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤.

    Returns:
        –°—Ç—Ä–æ–∫–∞ –¥–ª—è –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è.
  """
  return "{" + ", ".join(fruit_set) + "}"


# –°–æ–∑–¥–∞–µ–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ñ—Ä—É–∫—Ç–æ–≤
fruits_set_A = create_fruit_set("üçéüçê")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ A: {üçé, üçê}
fruits_set_B = create_fruit_set("üçêüçâ")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ B: {üçê, üçâ}
fruits_set_C = create_fruit_set("üçéüçêüçâ")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ C: {üçé, üçê, üçâ}
fruits_set_D = create_fruit_set("üß∫")  # –ú–Ω–æ–∂–µ—Å—Ç–≤–æ D: {üß∫}

# –í—ã–≤–æ–¥–∏–º –º–Ω–æ–∂–µ—Å—Ç–≤–∞
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ A: {display_set(fruits_set_A)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ B: {display_set(fruits_set_B)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ C: {display_set(fruits_set_C)}")
print(f"–ú–Ω–æ–∂–µ—Å—Ç–≤–æ D: {display_set(fruits_set_D)}")

# –û–±—ä–µ–¥–∏–Ω–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ (Union)
union_result = fruits_set_A | fruits_set_B
print(f"A ‚à™ B: {display_set(union_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé, üçê, üçâ}

# –ü–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ (Intersection)
intersection_result = fruits_set_A & fruits_set_B
print(f"A ‚à© B: {display_set(intersection_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçê}

# –†–∞–∑–Ω–æ—Å—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤ (Difference)
difference_result_AB = fruits_set_A - fruits_set_B
print(f"A - B: {display_set(difference_result_AB)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé}
difference_result_BA = fruits_set_B - fruits_set_A
print(f"B - A: {display_set(difference_result_BA)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçâ}

# –°–∏–º–º–µ—Ç—Ä–∏—á–Ω–∞—è —Ä–∞–∑–Ω–æ—Å—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤ (Symmetric Difference)
symmetric_difference_result = fruits_set_A ^ fruits_set_B
print(f"A ^ B: {display_set(symmetric_difference_result)}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: {üçé, üçâ}

# –ü–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Subset)
subset_result1 = fruits_set_A <= fruits_set_C
print(f"A <= C: {subset_result1}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True (A –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ C)
subset_result2 = fruits_set_A <= fruits_set_B
print(f"A <= B: {subset_result2}") # –†–µ–∑—É–ª—å—Ç–∞—Ç: False (A –Ω–µ –ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ B)

# –ù–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ (Superset)
superset_result1 = fruits_set_C >= fruits_set_A
print(f"C >= A: {superset_result1}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True (C –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ A)
superset_result2 = fruits_set_B >= fruits_set_A
print(f"B >= A: {superset_result2}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: False (B –Ω–µ –Ω–∞–¥–º–Ω–æ–∂–µ—Å—Ç–≤–æ A)

# –ü—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞ –Ω–∞–ª–∏—á–∏–µ —ç–ª–µ–º–µ–Ω—Ç–∞
print(f"üçé in A: {'üçé' in fruits_set_A}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: True
print(f"üçâ in A: {'üçâ' in fruits_set_A}")  # –†–µ–∑—É–ª—å—Ç–∞—Ç: False
```

    *   `set(fruit_string)` –ø—Ä–µ–≤—Ä–∞—â–∞–µ—Ç —Å—Ç—Ä–æ–∫—É –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —É–±–∏—Ä–∞—è –¥—É–±–ª–∏–∫–∞—Ç—ã –∏ –¥–µ–ª–∞—è –ø–æ—Ä—è–¥–æ–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –Ω–µ–≤–∞–∂–Ω—ã–º.
    *   –ú—ã –ø—Ä–æ–≤–µ—Ä—è–µ–º —á—Ç–æ —Å—Ç—Ä–æ–∫–∞ —Å–æ—Å—Ç–æ–∏—Ç —Ç–æ–ª—å–∫–æ –∏–∑ —Ä–∞–∑—Ä–µ—à–µ–Ω–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ —é–Ω–∏–∫–æ–¥–∞.

3.  **–ü—Ä–∏–º–µ—Ä—ã:**  –ú—ã —Å–æ–∑–¥–∞–µ–º –Ω–µ—Å–∫–æ–ª—å–∫–æ –º–Ω–æ–∂–µ—Å—Ç–≤ (`fruits_set_A`, `fruits_set_B`, `fruits_set_C` –∏ `fruits_set_D`) –∏ –ø—Ä–∏–º–µ–Ω—è–µ–º —Ä–∞–∑–ª–∏—á–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ –Ω–∏–º–∏.
